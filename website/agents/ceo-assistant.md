# Ohana CEO Assistant Agent — deprecated

This role is deprecated. Do not create new `reports/ceo/` reports.

Human/product feedback now belongs to the PMT workflow:

- Prompt: [`pmt.md`](pmt.md)
- Feedback reports: `reports/pmt/`
- Historical human/PMT feedback moved to: `reports/pmt/human-feedback/`
- Working model: [`../docs/working-style.md`](../docs/working-style.md)

If a task still asks for the CEO assistant, treat it as a PMT feedback-intake task and follow `agents/pmt.md`.
